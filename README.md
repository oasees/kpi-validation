# kpi-validation
Scripts and Github Actions workflows to validate part of the KPIs set for OASEES.

This repo currently covers the following KPIs:

- < 2 sec Service deployment time
- >= 3 distributed components of the same service
- >= 50 concurrent services running
- >= 50 edge nodes managed by the same OASEES instance
- Demonstration of multi-domain edge services with >= 3 OASEES instances
- Marketplace PoC with >= 50 OASEES deployments and capabilities adverstised
